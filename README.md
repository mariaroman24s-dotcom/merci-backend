# MERCI AI Platform - Backend

Backend API REST para la plataforma MERCI AI Platform, sistema SaaS multiempresa para gestión de agentes virtuales, extensiones, llamadas, usuarios, configuración PBX y sincronización con Cloud UCM.

## Tecnologías

- Node.js
- Express
- PostgreSQL
- JWT
- bcrypt
- Axios
- Cloud UCM API
